# Personal Academic Homepage

This is a personal academic homepage built with React, Tailwind CSS, and Vite.

## Features

- **Profile**: Introduction, avatar, social links.
- **Publications**: List of papers and projects.
- **Blog**: Markdown-based blog system.
- **About**: Detailed biography and contact info.
- **Responsive**: Works on desktop and mobile.
- **Academic Style**: Clean, Palatino font, professional layout.

## Getting Started

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Start development server**:
    ```bash
    npm run dev
    ```

3.  **Build for production**:
    ```bash
    npm run build
    ```

## Customization

-   **Profile**: Edit `src/data/profile.json`
-   **Publications**: Edit `src/data/publications.json`
-   **Projects**: Edit `src/data/projects.json`
-   **Blog Posts**: Add `.md` files to `src/posts/`
